# Roteiro de Demonstração - Escola Van (2-3 Minutos)

Este roteiro foi planejado para um vídeo curto e objetivo, destacando as principais funcionalidades e a modernização do sistema "Escola Van".

---

## 1. Introdução e Branding (0:00 - 0:20)
**Visual:** Tela de Login (Logo "Escola Van" em destaque).
**Narração:**
"Bem-vindo ao **Escola Van**, a solução completa para gestão de transporte escolar. Recentemente, passamos por um rebranding completo, modernizando nossa identidade visual e interface para oferecer uma experiência mais amigável e profissional."

## 2. Acesso e Segurança (0:20 - 0:40)
**Visual:** Preenchimento de login e acesso ao sistema.
**Ação:** Fazer login como Administrador.
**Narração:**
"A segurança é nossa prioridade. O sistema conta com autenticação robusta via JWT e criptografia de senhas. Ao acessar, o administrador é recebido por um painel intuitivo e seguro."

## 3. Dashboard Administrativo (0:40 - 1:10)
**Visual:** Dashboard Principal (Cards de KPIs, Gráfico de Faturamento).
**Ação:** Passar o mouse sobre os cards e o gráfico.
**Narração:**
"No Dashboard, temos uma visão geral do negócio em tempo real.
*   **KPIs:** Total de Responsáveis, Pagamentos Pendentes e Receita Mensal.
*   **Gráficos Interativos:** Acompanhamento visual da previsão de faturamento.
Tudo desenhado com um layout moderno (Dark Mode) para facilitar a leitura."

## 4. Gestão Financeira e Contratos (1:10 - 1:40)
**Visual:** Seção de Pagamentos/Responsáveis.
**Ação:** Expandir um Responsável, mostrar lista de pagamentos e status.
**Narração:**
"A gestão financeira é simplificada.
*   Organizamos pagamentos por Responsável e Contrato.
*   Visualizamos status de pagamentos (Pendente, Pago, Atrasado) com indicadores claros.
*   O sistema permite baixa manual e controle detalhado de mensalidades."

## 5. Assistente Virtual: Jaspion (1:40 - 2:10)
**Visual:** Clicar no ícone do Jaspion (canto inferior direito).
**Ação:** Abrir chat, enviar uma mensagem (ex: "Como altero minha senha?").
**Narração:**
"Para suporte rápido, integramos o **Jaspion**, nosso assistente virtual inteligente.
*   Ele responde dúvidas frequentes sobre cadastro, pagamentos e uso do sistema.
*   O ícone e a interface foram personalizados para criar uma identidade única e nostálgica, tornando o suporte mais leve e acessível."

## 6. Visão do Cliente e Responsividade (2:10 - 2:30)
**Visual:** Alternar para visão mobile (ou simular tela menor).
**Narração:**
"O Escola Van é totalmente responsivo. Pais e responsáveis podem acessar seus painéis pelo celular para verificar rotas, pagamentos e comunicados, garantindo transparência e facilidade no dia a dia."

## 7. Tecnologias Utilizadas (2:30 - 2:45)
**Visual:** Slide com lista de tecnologias ou tela "Sobre".
**Narração:**
"O projeto foi construído atendendo rigorosamente aos requisitos técnicos:
*   **Planejamento:** Levantamento detalhado de requisitos funcionais e não funcionais.
*   **Interface:** Prototipagem e design responsivo com HTML5 e CSS3.
*   **Front-End:** JavaScript dinâmico consumindo APIs RESTful.
*   **Back-End:** API em Node.js e TypeScript, com arquitetura SOLID e Swagger.
*   **Dados:** Banco de dados relacional (MySQL) integrado.
*   **Inovação:** Chatbot com Machine Learning para suporte inteligente."

## 8. Conclusão (2:45 - 3:15)
**Visual:** Volta para o Dashboard ou Tela de "Sobre".
**Narração:**
"Com tecnologia de ponta, design focado no usuário e ferramentas poderosas de gestão, o **Escola Van** transforma a administração do transporte escolar, permitindo que você foque no que mais importa: a segurança e o transporte das crianças.
Obrigado por assistir!"

---

**Notas Técnicas para a Gravação:**
*   **Resolução:** Gravar em 1080p ou superior.
*   **Audio:** Usar microfone de boa qualidade, ambiente silencioso.
*   **Navegação:** Movimentos de mouse suaves, sem cliques bruscos.
*   **Zoom:** Usar zoom na edição para destacar áreas de interesse (ex: gráfico, chat).
